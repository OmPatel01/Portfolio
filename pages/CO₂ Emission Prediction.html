Project Overview

This project focuses on predicting hourly CO₂ emissions in steel production, allowing businesses to monitor and optimize their processes to reduce environmental impact. By leveraging machine learning models, stakeholders can proactively manage emissions, meet regulatory standards, and enhance sustainability in production operations. The project initially utilized Polynomial Regression for prediction and was later deployed using Flask and Docker, making it accessible via an API on Render.

Key Insights

  • Accurate hourly predictions help industries optimize production schedules to reduce emissions.
  • Polynomial Regression (degree 2) captures non-linear emission patterns better than linear models.
  • PCA (Principal Component Analysis) aids in feature reduction and better visualization of emissions.
  • Deploying the model as an API makes predictions accessible for integration into industrial systems.

Technical Implementation

  • Data Preprocessing:
    • Aggregated data to hourly values for accurate forecasting.
    • Applied PCA to reduce dimensions and identify emission patterns.
  • Modeling Approach:
    • Started with Linear Regression but observed non-linearity.
    • Implemented Polynomial Regression (degree 2), significantly improving accuracy.
  • Deployment & MLOps:
    • Built a Flask API to serve predictions.
    • Containerized the model with Docker for consistent deployment.
    • Deployed on Render, making it publicly accessible.
    • Previously developed a Streamlit app for real-time CO₂ monitoring.
